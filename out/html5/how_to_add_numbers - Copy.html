How to add numbers

Instructions for using the addition function on the JPS Calculator app.

To succesfully add numbers, you must understand the definition of addition.

  1. Enter the first number by pressing it.
  2. Press the minus [+] button
  3. Enter the second number by pressing it.
  4. Press the equals [=] button.
The sum of the equation is displayed on the screen.
The first number [8] plus [+] the second number [5] equals [=] 13.

8 + 5 = 13
