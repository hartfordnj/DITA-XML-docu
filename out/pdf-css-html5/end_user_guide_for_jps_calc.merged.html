Sum Up

Addtion

the action or process of adding something to something else.

Addition is adding number(s) or value(s) to another number or value.

If we begin with the number [8] and add [5] to it, we are left with [13].

In the JPS Calculator App the addition fucntion is represented by the [+] key.

Example: 8 + 5 = 3

How to add numbers

Instructions for using the addition function on the JPS Calculator app.

To succesfully add numbers, you must understand the definition of addition.

  1. Enter the first number by pressing it.
  2. Press the minus [+] button
  3. Enter the second number by pressing it.
  4. Press the equals [=] button.
The sum of the equation is displayed on the screen.
The first number [8] plus [+] the second number [5] equals [=] 13.

8 + 5 = 13

Subtract

Subtraction

the process or skill of taking one number or amount away from another.

Subtraction is removing one number or value from another.

If we begin with the number [8] and remove (subtract) [5], we are left with [3].

In the JPS Calculator App the subtracting fucntion is represented by the [-] key.

Example: 8-5=3

Multiply

Multiply

obtain from (a number) another that contains the first number a specified number of times.

To multiply a number we take an established number and repeat it as many times as determined by the second number.

If we begin with the number [8] and repeat it (multiply) [5] times (8+8+8+8+8), we are left with [40].

In the JPS Calculator App the multiply fucntion is represented by the [x] key.

Example: 8 x 5 = 40

Divide

Divide

The method for dividing large numbers into smaller groups or parts.

To divide a number we take an established number and seperate it as many times as determined by the second number.

If we begin with the number [40] and separate it (divide) into [5] equal parts, we are left with [8] parts.

In the JPS Calculator App the divide fucntion is represented by the [/] key.

Example: 40 / 5 = 8
